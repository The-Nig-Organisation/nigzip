Utiliser NIGZIP:

1) Déposer le fichier a compresser dans le dossier de NIGZIP
2) Changer la variable "filename" dans "options.ini"
3) Changer la variable "finalname" par le nom du fichier qui sera généré
3.1 OPTIONNEL) Changer la variable "percentage" par le pourcentage désiré
3.2 OPTIONNEL) Changer la variable "starting_byte" par la valeur désiré
3.3 OPTIONNEL) Changer la seed par la valeur désiré
4) Exécuter le programme
5 OPTIONNEL) Compresser le fichier généré avec gzip ou un autre algorithme de compression
